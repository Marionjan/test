Set-Top Box Multimédia

Introduction

Le but de ce projet est de créer l'ébauche d'un logiciel pour une set-top box multimédia permettant de jouer des vidéos, des films, d'afficher des photos, et plus encore. Ce projet est divisé en deux parties principales :

  • Serveur C++ : Le serveur gère la communication réseau et le traitement des demandes de clients via des sockets.
  • Client Java : Le client est une application avec une interface graphique qui permet à l'utilisateur de se connecter au serveur, de rechercher et de jouer des vidéos, de visualiser des photos, etc.

Prérequis

Serveur C++ :

  • Système d'exploitation : Linux / Windows
  • Compilateur C++ (g++ recommandé)
  • Libs pour la gestion des sockets (par défaut, les sockets sont gérés par les librairies systèmes)

Client Java :

  • Système d'exploitation : Linux / Windows
  • Java 8 ou version supérieure
  • JDK installé (Java Development Kit)

Compilation et exécution

1. Serveur C++ (avec Makefile)

Le serveur est écrit en C++ et utilise des sockets pour la communication. Il y a un Makefile pour la partie serveur.

Étapes pour compiler et exécuter le serveur :

  1. Ouvrez un terminal dans le répertoire où se trouve le Makefile du serveur.
  2. Exécutez la commande suivante pour compiler et exécuter le serveur : 
    make run
  3. Cette commande va d'abord compiler le code C++ à l'aide du Makefile, générer l'exécutable du serveur, puis lancer le serveur. Le serveur commencera à écouter sur le port 3331 (ou le port configuré dans le code).

2. Client Java (avec Makefile)

Le client est écrit en Java et utilise Java Swing pour l'interface graphique. Il y a également un Makefile pour la partie cliente Java.

Étapes pour compiler et exécuter le client :

  1. Ouvrez un terminal dans le répertoire où se trouve le Makefile du client.
  2. Exécutez la commande suivante pour compiler et exécuter le code Java : 
    make run
  3. Cette commande va d'abord compiler les fichiers Java et générer les fichiers .class, puis lancer l'application Java.

Connexion au Serveur

Assurez-vous que le serveur est en cours d'exécution sur l'adresse localhost (ou modifiez l'adresse IP dans le code si nécessaire). L'application cliente tentera de se connecter au serveur à l'adresse spécifiée (localhost:3331 par défaut).

Utilisation

  • Lancer l'application : Ouvrez l'application cliente Java (MainWindow).
  • Rechercher un objet multimédia : Entrez un nom d'objet dans le champ de saisie et cliquez sur "Rechercher".
  • Jouer un objet multimédia : Sélectionnez un objet multimédia recherché et cliquez sur "Jouer".
  • Créer un objet multimédia : Cliquez sur "Créer" pour ouvrir un formulaire permettant de créer un nouvel objet multimédia.

Technologies utilisées

  • Java Swing pour l'interface graphique.
  • C++ (Sockets) pour la gestion des connexions réseau.
  • TCP/IP pour la communication entre le client et le serveur.
