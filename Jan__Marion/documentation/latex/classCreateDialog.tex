\doxysubsection{Detailed Description}
Represents a dialog window for creating a multimedia object. 

Represents a dialog window to create a multimedia object (photo, video, or film). The dialog includes fields for entering the name, path, and type-\/specific properties (e.\+g., dimensions, duration, chapters).

This class communicates with the server through a \mbox{\hyperlink{classClient}{Client}} object.
