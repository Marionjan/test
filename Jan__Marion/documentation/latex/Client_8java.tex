\mbox{\hyperlink{classClient}{Client}} Java program to communicate with the C++ server. 

This class establishes a connection to a C++ server, sends requests, and handles responses. It reads user input from the terminal, sends requests to the server, and displays the server\textquotesingle{}s responses.
