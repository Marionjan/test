\doxysubsection{Detailed Description}
Action to exit the application. 

This class defines an action to quit the application when triggered. It extends \mbox{\hyperlink{}{Abstract\+Action}} and overrides the \mbox{\hyperlink{}{action\+Performed}} method to call \mbox{\hyperlink{}{System\#exit}} with a status code of 0, terminating the program.
