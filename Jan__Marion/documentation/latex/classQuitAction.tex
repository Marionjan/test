\doxysubsection{Detailed Description}
Represents an administrator in the system. 

Action to quit the application.

This class implements an action that can be assigned to a button or menu item to close the application. When the action is triggered, the \mbox{\hyperlink{classQuitAction_a97131ce828e2a8ae9691d6fdecf753a0}{action\+Performed}} method is called, which terminates the program.
